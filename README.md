# Praneet Gogoi — AI/ML Engineer & Manga-Themed Portfolio v1

A dynamic, interactive, manga-themed personal portfolio website built with HTML5, CSS3, and modern JavaScript. Featuring custom animations, a terminal console, command palette (`⌘K`), sound effects typography, and responsive manga panel layouts.

## 🌟 Features

- **Manga Panel Layout**: Comic-book aesthetic with halftone backgrounds, speech bubbles, and dynamic action panels.
- **Interactive Terminal**: Simulated console with tabs for AI/ML training logs, deployment statuses, and CRT screen effects.
- **Command Palette (`⌘K`)**: Quick navigation search overlay allowing users to jump directly to sections or open actions.
- **Embedded Resume Modal**: View the complete resume directly inside the browser in a stylized pop-up overlay.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices with fluid typography and touch accessibility.

## 🚀 Deployment & Usage

Open `index.html` directly in any web browser, or host via **GitHub Pages**:
1. Go to repository **Settings** > **Pages**.
2. Select `main` branch and `/ (root)` folder.
3. Click **Save**. Your site will be live at `https://praneetgogoi.github.io/Praneet-gogoi-v1/`.

## 📁 Files Overview

- `index.html` / `praneet_manga_portfolio.html`: Main portfolio application file containing HTML structure, design tokens, styling, and interactive scripts.
- `Praneet_Gogoi_Resume.pdf`: Integrated resume PDF viewed via the modal overlay.
